# Contacts

## 주제
아주 간단한 연락처 기능

## 전체적인 기능 설명
간단한 연락처를 관리한다. 여기서 간단한 연락처는 이름과 전화번호로 구성되어 있는 연락처를 말한다.
CRUD기능을 이용해 기본적인 연락처 조회, 추가, 수정, 삭제를 할 수 있으며 추가로 이름 검색과 번호 검색으로 연락처를 찾는 기능도 가능하도록 구현할 것이다.
